# 3-day-beginner-angular-project
The challenge is that you have no angular experience and must make an angular app from scratch in three days.

## Goal: Build a small web application with a login system.
For this project you will need to create Front End using Angular, API using C# and a database using SQL Server.

### Database Instructions
* Create a database called Portal that you will use with this project.
* Create a Users table that has at least 2 main fields (Email and Password). Feel free to create any other necessary fields.
* Make sure that the data stored in the password field is encrypted.

### API Instructions
* Create the API utilizing C# and best practices.
* Create endpoints for the login system that can do user validations.

### Front End Instructions
* Create a front end utilizing the newest version of Angular.
* Create a login page that as a place to input an email, and a password.
* Please be sure to include any error handling that you would expect to see in a real world application.
